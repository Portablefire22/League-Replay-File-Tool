# League Replay File (.LRF) Tool 

This is a tool that'll evolve with my needs, so far it only categorises files into separate 
folders by their client version. All metadata for these files gets saved into a single JSON 
file.

# Working Data

These files aren't exactly common any more, so I can only test with an archive I found.

[The download for the archive can be found here](https://archive.org/details/lrpkts_420.7z)
